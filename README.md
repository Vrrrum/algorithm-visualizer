# Prosty wizualizator algorytmów  

## Instalacja

### 1. Zklonj repozytorium
```Bash
git clone https://github.com/Vrrrum/algorithm-visualizer.git
```

### 2. Zainstaluj pakiety
```Bash
npm install
```

### 3. Rozpocznij program
```Bash 
npm run dev
```
